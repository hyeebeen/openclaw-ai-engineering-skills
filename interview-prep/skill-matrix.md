## 学习建议

### 3 个月速成计划

```
Week 1-4: 基础夯实
├── Python 高级编程
├── 数据结构复习
└── 完成 2 个基础项目

Week 5-8: AI 核心技能
├── RAG 系统构建（重点）
├── Prompt 工程
└── 完成 2 个 AI 项目

Week 9-12: 进阶与面试
├── Agent 设计
├── 系统设计练习
├── 模拟面试
└── 完善作品集
```

### 6 个月深度学习计划

```
Month 1-2: 基础 + RAG
Month 3-4: Agent + Function Calling
Month 5: 评估 + 部署
Month 6: 综合项目 + 面试准备
```

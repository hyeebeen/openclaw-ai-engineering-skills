## 入门级项目 (⭐)

### 1. 个人文档问答助手

**描述**: 构建一个能回答关于你个人文档（笔记、PDF、Word）问题的 AI 助手

**核心技能**: RAG 基础、文档处理

**技术栈**: 
- Chroma（向量数据库）
- OpenAI Embedding + GPT
- Python + Streamlit（前端）

**功能清单**:
- [ ] 上传 PDF/Markdown 文档
- [ ] 自动分块和嵌入
- [ ] 自然语言问答
- [ ] 显示答案来源

**扩展挑战**:
- 支持多轮对话
- 添加文档管理界面
- 支持多种文件格式

## 项目展示建议

### GitHub 仓库结构

```
project-name/
├── README.md          # 项目说明（最重要！）
├── demo.gif           # 演示动图
├── architecture.png   # 架构图
├── src/               # 源代码
├── tests/             # 测试
├── docs/              # 详细文档
└── examples/          # 使用示例
```

### README 必备要素

1. **30 秒理解**: 项目是什么、解决什么问题
2. **快速开始**: 5 分钟可运行
3. **演示**: GIF 或截图
4. **技术栈**: 清晰列出
5. **架构图**: 视觉化展示
6. **核心功能**: 功能清单
7. **部署指南**: 如何部署

### 面试展示技巧

1. **讲故事**: 为什么做这个项目、解决了什么问题
2. **展示思考**: 技术选型理由、权衡考虑
3. **量化成果**: 性能指标、用户反馈
4. **承认不足**: 当前局限、未来改进
